Games

I have a lot of things planned, and this is where I will be putting said things.

UPDATE: For the first early beta (which is entirely FREE!) I have a release date set for June 1st.

90 Degrees is a game where you play as a red square jumping on grey platforms.

This was my first ever game idea, and a rough proof of concept called This is You was released on June 6th, 2023. 90 Degrees is my attempt to fully realize the idea into a longer, steam-worthy game, that will contain many new levels, new game mechanics, a lot more polish, and a fully featured level editor.

Just like This is You, 90 Degrees will not be an easy game to complete (actually, it will probably be much, MUCH harder to 100% than This is You)

Expect a demo sometime soon! As soon as a Steam page is created for the game, I will link that here.
